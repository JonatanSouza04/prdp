# Dependencias 
  - express
  - express-session
  - jsonwebtoken (se for realizar o Login)

# Descrição
  Esse backend é responsável por realizar toda a regra de negócio da Sorvetes Gelados, como cadastrar os produtos, funcionários e registrar suas vendas.

# Instalação

  -  npm install
